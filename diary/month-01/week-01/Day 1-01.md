# Day 1: Python 热身——从 Java 思维切换到 Python

**日期**：2026-04-21

## 今天做了什么

花了半天时间把 Python 环境搭好，然后重点对比了 Python 和 Java 在类型系统、异步模型上的差异。

**类型注解**：`def greet(name: str) -> str:` 这种写法熟悉又陌生。Java 的泛型用了这么多年，但 Python 的类型注解是 optional 的，一开始总不习惯加。

**async/await**：Java 的 CompletableFuture 和 Python 的 asyncio 思路很像，但 Python 的写法更简洁。写了一个简单的异步 HTTP 请求 demo，对比了同步和异步的耗时。

**pydantic**：太像 Java 的 Bean Validation 了，有亲切感。用 BaseModel 定义了一个数据模型，顺手写了字段校验。

**dotenv**：管理 API Key 的方式。从这个角度来说，Python 生态工具的简洁度确实比 Java 高不少。
