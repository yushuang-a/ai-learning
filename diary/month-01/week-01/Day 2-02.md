# Day 2: 首次调用 LLM API

**日期**：2026-04-22

## 今天做了什么

终于第一次调通了 LLM API。客户端的初始化方式和 Java 的 HttpClient 差不多，但返回结果处理起来更灵活。

写了一个 `simple_chat()` 函数，传入 user message 和可选的 system prompt，返回 LLM 的回答。调了 `gpt-4o-mini` 模型，响应速度很快。

**关键发现**：System Prompt 的效果比想象中更明显。同样的 user message，system prompt 写成"你是一个严格的代码评审专家"和"你是一个友好的学习伙伴"，回复风格完全不同。这让我意识到 Prompt 本身就是一种编程——用自然语言控制模型行为。

代码放在 `code/01-llm-basics/01_chat_completion.py`。
